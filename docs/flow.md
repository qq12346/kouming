## 核心流程：从"有一个意图"到"审视多 Agent 协作结果"

## 关键决策点

| DP | 位置 | 默认行为 | 用户操作 |
|----|------|---------|---------|
| 跳过溯源追问 | 意图提交后 | 必须展示，不可默认跳过 | 主动跳过 |
| 修改价值观 | 溯源追问后 | 同意系统推断 | 逐条修改 |
| 自己做某步 | Agent 执行中 | AI 自动 | 随时开关 |
| 反驳 AI 假设 | 结果展示后 | 显示，不强制反驳 | 逐条反驳 |
| Shell 命令确认（v0.3） | Executor 执行前 | 每条命令需确认 | 确认/拒绝/预览 |

---

## 异常流程

- **API Key 未配置** → 弹窗引导配置
- **API 调用失败** → 展示错误 + 重试 + 保留已填内容
- **宪法过滤器阻断** → 自动补全 + 标注"[系统补全]" + 用户可编辑
